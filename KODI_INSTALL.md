# Sabri Maranan Subtitles Kodi Addon - Installation Guide

## Quick Install

### For Kodi Users

1. Open **Kodi Settings** → **Add-ons** → **Install from repository**
2. Add new repository with this URL:
   ```
   https://motekamazon-droid.github.io/repo/
   ```
   (Shortened: `bit.ly/sabri-addon`)
3. Browse to **Sabri Maranan Subtitles** and install
4. Enable the addon in **Settings** → **Add-ons** → **Services**
5. Play any Sabri Maranan episode — subtitles auto-load!

## What It Does

- **Auto-detects** Sabri Maranan episodes (any filename with episode numbers)
- **Fetches subtitles** from GitHub (no login needed, public repo)
- **Automatically applies** subtitles when episode starts
- **Caches** subtitles for fast repeat playback
- **No popups** — silent, seamless operation

## Supported Episodes

Currently: **Episodes 1-17** (as of April 2, 2026)
Updates automatically as new episodes are processed

## Filename Requirements

Works with these patterns:
- `sabri_maranan_s1e12`
- `S01E12`, `s1e12`
- `Season 1 Episode 12`
- `Ep12`, `Episode 12`

## Troubleshooting

- **No subtitles loading?** Check Kodi logs (Settings → System → Logging)
- **Connection issues?** Verify internet connection
- **Episode not found?** Check if it's in the GitHub repo (episodes 1-17 available)
- **Can't find addon in repository?** Make sure the repository URL is entered correctly in step 2

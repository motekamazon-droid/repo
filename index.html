📦 Sabri Maranan Subtitles Repository

Kodi addon repository for auto-fetching English subtitles

Repository Files:

📁 zips/ Browse
📄 addons.xml View
📄 addons.xml.md5 View

Installation:

  1. In Kodi: Settings → File Manager → Add source
  2. Enter this repository URL
  3. Settings → Add-ons → Install from ZIP
  4. Navigate to zips/service.sabri-maranan-subtitles/
  5. Select the addon ZIP and install
